### Fedora (dnf)

#### Commands Practiced
- dnf install
- dnf remove
- dnf update
- dnf search
- dnf info

#### DevOps Context
On Fedora/RHEL-based systems, `dnf` is used to:
- Install application dependencies
- Apply security updates
- Manage system packages on servers
- Automate provisioning using scripts and configuration tools

### Arch Linux (pacman)

#### Commands Practiced
- pacman -S
- pacman -R
- pacman -Syu
- pacman -Ss
- pacman -Qi

#### DevOps Context
On Arch Linux, `pacman` is used to:
- Manage lightweight and minimal server environments
- Keep systems up to date with rolling releases
- Install only required components for optimized setups
- Build highly controlled DevOps and automation environments
