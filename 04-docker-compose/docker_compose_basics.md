## Docker Compose Basics (DevOps Perspective)

### What is Docker Compose?
Docker Compose is a tool to define and run multi-container Docker applications using a YAML file.

### Why Docker Compose?
- Run multiple services together
- Define networking and volumes declaratively
- Simplify local development and testing
- Reproduce environments easily

### Key Concepts
- services → containers
- volumes  → persistent storage
- networks → service communication

### DevOps Context
Docker Compose is commonly used for:
- Local development environments
- Integration testing
- CI pipeline service dependencies
