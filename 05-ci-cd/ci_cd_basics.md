## CI/CD Basics (DevOps Perspective)

### What is CI?
Continuous Integration automatically:
- Builds code
- Runs checks on every push or pull request
- Detects errors early
- Ensures code quality and consistency

### What is CD?
Continuous Deployment/Delivery automatically:
- Preparing application releases
- Deploys applications (optional/manual)

### DevOps Context
CI/CD pipelines:
- Reduce manual work
- Enforce standards automatically
- Provide fast feedback to developers
- Increase reliability
- Ensure consistent delivery
